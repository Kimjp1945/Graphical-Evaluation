CCCGraphEval.m is developed to conduct the graphical evaluation based on CCC, proposed by Dr. Lin, LK.
This program calls CCC.m to compute the CCC value.

Two real examples are considered in the manuscript (see Real Data Analysis for Manuscript.m). One data comes from B-A manuscirpt and the other is Radiomics data.
real_example_data.xlsx includes those data set in both sheets. Only two Radiomics data are included. Full data sets are availabel upon resquest.
